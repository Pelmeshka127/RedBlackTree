# RedBlackTree

## Установка и запуск проетка
Перед установкой проекта убедитесь, что у вас установлена библиотека [graphviz](https://graphviz.org).

Для установки
```
git clone git@github.com:Pelmeshka127/RedBlackTree.git

cd RedBlackTree
```

Для запуска
```
cmake -B build

cd build/src/

cmake --build .

./rbt
```
